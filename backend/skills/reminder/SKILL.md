# Reminder Skill

你可以帮助用户管理提醒事项。

## 能力

- 创建一次性提醒
- 查看提醒列表
- 取消提醒

## 使用方式

当用户请求提醒时：

1. **创建提醒**：提取内容和时间，调用 `reminder` 工具
   - 用户: "提醒我明天下午3点开会"
   - 调用: reminder(action="create", content="开会", time="明天下午3点")

2. **查看提醒**：当用户询问有哪些提醒
   - 用户: "我有哪些提醒"
   - 调用: reminder(action="list")

3. **取消提醒**：当用户要取消某个提醒
   - 用户: "取消开会的提醒"
   - 调用: reminder(action="cancel", content="开会")

## 时间解析

支持自然语言时间：
- "X分钟后" - 相对时间
- "明天X点" - 绝对时间
- "下周X" - 周计算
- "今晚X点" - 当天时间

## 注意事项

- 确认时间解析正确后再创建提醒
- 创建成功后简洁确认
- 如果时间不明确，询问用户
