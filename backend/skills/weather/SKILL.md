# Weather Skill

帮助用户查询天气信息。

## 能力

- 查询主客户端位置的天气
- 支持查询今天、明天、后天的天气
- 发送天气通知到指定客户端

## 使用方式

1. **查询今天天气**：
   - 用户: "今天天气怎么样"
   - 调用: weather_get(day="今天")

2. **查询明天天气**：
   - 用户: "明天天气怎么样"
   - 调用: weather_get(day="明天")

3. **查询后天天气**：
   - 用户: "后天天气"
   - 调用: weather_get(day="后天")

4. **推送天气**：
   - 用户: "推送天气给所有设备"
   - 调用: weather_send()

## 参数说明

### weather_get
- `day` (可选): 查询哪天的天气，可选值：今天、明天、后天。默认今天

### 返回值
- `location`: 位置
- `day_label`: 今天/明天/后天
- `date`: 日期
- `temp_max`: 最高温度
- `temp_min`: 最低温度
- `text_day`: 白天天气
- `text_night`: 夜间天气
- `message`: 完整的天气描述
- `all_days`: 所有三天的天气数据

## 注意事项

- weather_get 会自动使用主客户端的位置信息
- 主客户端超过24小时未更新位置时，会降级使用第一个在线客户端的位置
- weather_send 会发送给所有启用天气推送的客户端
